process method

Future<BearContext> process (BearContext c)

Execute each BearMiddleware in middleware, then return the modified ( or not ) BearContext in a Future.

Implementation 

Future<BearContext> process(BearContext c) async {
  for (var middleware in middlewares) {
    c = await middleware.process(c);
  }

  // Automatically transformed to Future<BearContext>
  return c;
}
